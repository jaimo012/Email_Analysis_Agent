# 📧 이메일 AI 초안 자동화 (Email AI Draft Agent)

사용자의 실제 이메일 작성 스타일을 학습하여, 수신된 메일을 분류하고 최적화된 답장 초안을 생성하는 스마트 Gmail 관리 시스템입니다. Google Apps Script(GAS)를 기반으로 작동하며, Claude 4.5 모델의 지능형 라우팅을 통해 효율적인 업무 자동화를 구현합니다.

## 🎯 주요 목표
* **개인화된 초안 생성**: 사용자의 과거 발신 메일 200개를 분석하여 고유의 말투, 인사말, 서명을 완벽히 재현합니다.
* **지능형 모델 라우팅**: 작업 난이도에 따라 Sonnet 4.5와 Haiku 4.5 모델을 분배하여 비용과 속도를 최적화합니다.
* **리크루팅 전문가 모드**: 프로젝트 제안 및 소싱 메일은 '10년 차 수석 리크루터' 페르소나를 통해 심층 질문이 포함된 전문 답장을 작성합니다.
* **업무 시간 혁신**: 반복적인 이메일 확인 및 분류 시간을 획기적으로 단축하여 핵심 업무에 집중할 수 있도록 돕습니다.

## 🛠 기술 스택 및 모델 구성
* **Engine**: Google Apps Script (GAS)
* **AI Models (Claude 4.5)**:
    * **Sonnet 4.5** (`claude-sonnet-4-5-20250929`): 고도의 문맥 파악 및 자연스러운 답장 생성 (초안 작성, 스타일 분석 전담)
    * **Haiku 4.5** (`claude-haiku-4-5-20251001`): 초고속 텍스트 스캐닝 및 카테고리 분류 (스팸 판별, 유형 분류 전담)
* **Integrations**: Gmail API, Slack Webhook, Google Sheets

## 📂 파일 구조 및 역할
| 파일명 | 역할 설명 |
| :--- | :--- |
| `1_Config.gs` | API 키, 슬랙 URL, 시트 ID, 모델 식별자 등 시스템 전역 설정 관리 |
| `0_UpdateSheet.gs` | 데이터베이스(구글 시트) 탭 생성 및 헤더 초기 세팅 유틸리티 |
| `2_Step0_StyleAnalysis.gs` | 최근 발신 메일 학습을 통한 '사용자 스타일 가이드' 생성 및 저장 |
| `3_Main_Routine.gs` | 30분 주기 트리거 실행. 메일 수집, 유형 분류, 라벨링 등 전체 흐름 제어 |
| `4_AI_Service.gs` | Claude API 통신 및 카테고리별(일반/소싱/스팸) 맞춤형 프롬프트 제어 |
| `5_External_Service.gs` | 슬랙 메시지 전송 및 구글 시트(스팸 기록, 프롬프트 피드백) 데이터 적재 |

## 🔄 시스템 워크플로우

### 1. 스타일 가이드 수립 (Step 0)
* 최근 3개월간 보낸 메일을 분석하여 수신 유형별 대응 방식과 나만의 문체를 파악합니다.
* 분석 결과는 시트의 `스타일_가이드` 탭에 기록되어 시스템의 기본 지침으로 활용됩니다.

### 2. 정밀 메일 수집 및 필터링
* **수집 조건**: 2026-04-01 이후 수신된 미읽음 메일 중 프로모션 및 소셜 카테고리는 자동으로 제외합니다.
* **숨은 참조(BCC) 탐지**: 수신인에 본인 주소가 없더라도, 회사 도메인(`kmong.com`)이 수신/참조인 목록에 없는 경우 숨은 참조로 판단하여 처리 대상에 포함합니다.

### 3. 지능형 분류 및 처리
* **Haiku 분류**: 모든 메일을 `SPAM`, `SOURCING`, `NORMAL` 3단계로 초고속 분류합니다.
* **Spam**: 시트에 로그를 남기고 즉시 종료하여 리소스를 절약합니다.
* **Sourcing**: `sourcing` 전용 라벨을 부착하고, 프로젝트의 핵심을 짚는 심층 질문이 포함된 리크루터 모드 답장을 생성합니다.
* **Normal**: 사용자 스타일 가이드를 준수하는 정중한 일반 비즈니스 답장을 생성합니다.

### 4. 슬랙 실시간 보고
* 생성된 모든 초안은 슬랙 DM으로 즉시 전송됩니다.
* 메시지 하단에는 해당 Gmail로 바로 연결되는 딥링크가 포함되어 즉시 발송이 가능합니다.

## ⚠️ 운영 제약 조건
* **Anti-AI 톤앤매너**: AI가 작성한 티가 나지 않도록 초안 내 마크다운 기호(`###`, `**` 등) 사용을 엄격히 금지합니다.
* **안정성**: GAS의 6분 실행 제한을 준수하기 위해 한 회차당 최대 10개의 메일만 순차적으로 처리합니다.
* **보안**: 모든 민감한 키(API Key, Webhook URL)는 스크립트 속성(Script Properties)에 안전하게 보관합니다.
