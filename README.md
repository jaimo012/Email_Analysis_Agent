# 📧 이메일 AI 초안 자동화 (Email AI Draft Agent)

사용자의 이메일 스타일을 학습한 Claude 4.5 기반의 Gmail 관리 및 답장 초안 자동 생성 시스템입니다. [cite_start]GAS(Google Apps Script)를 통해 주기적으로 실행되며, 슬랙(Slack)으로 업무 브리핑 및 AI 초안을 전달합니다. [cite: 77, 78]

## 🎯 주요 목표
* [cite_start]**개인화된 답장**: 사용자의 과거 발신 메일을 분석하여 말투, 인사말, 서명을 그대로 재현합니다. [cite: 80, 93]
* **지능형 모델 라우팅**: 고품질 작업은 Sonnet 4.5, 빠른 분류는 Haiku 4.5 모델을 사용하여 효율성을 극대화합니다.
* **전문가 모드 (Sourcing)**: 프로젝트 제안이나 인력 요청 메일은 '10년 차 수석 리크루터' 페르소나를 장착하여 핵심 질문이 포함된 전문적인 초안을 작성합니다.
* [cite_start]**업무 시간 단축**: 하루 1~2시간의 메일 확인 시간을 20분 이내로 단축합니다. [cite: 74]

## 🛠 기술 스택 및 모델 구성
* [cite_start]**Engine**: Google Apps Script (GAS) [cite: 74]
* **AI 모델 (Claude 4.5)**:
    * **Sonnet 4.5** (`claude-sonnet-4-5-20250929`): 스타일 분석 및 정교한 답장 초안 작성 전담
    * **Haiku 4.5** (`claude-haiku-4-5-20251001`): 스팸 판별 및 이메일 카테고리 3단 분류 전담
* [cite_start]**연동**: Gmail, Slack Webhook, Google Sheets [cite: 178-181]

## 📂 파일 구조 및 역할
| 파일명 | 역할 설명 |
| :--- | :--- |
| `1_Config.gs` | API 키, 슬랙 URL, 시트 ID, 모델명 등 핵심 상수 관리 |
| `0_UpdateSheet.gs` | 데이터베이스(구글 시트) 탭 생성 및 헤더 초기 세팅 |
| `2_Step0_StyleAnalysis.gs` | 최근 3개월 발신 메일을 분석하여 '나의 스타일 가이드'를 추출 및 저장 |
| `3_Main_Routine.gs` | 30분 단위 트리거 실행. 메일 수집, 분류, 처리 흐름 제어 |
| `4_AI_Service.gs` | 모델 라우팅 로직 및 유형별(일반/소싱) 전문 프롬프트 제어 |
| `5_External_Service.gs` | 슬랙 메시지 전송 및 구글 시트 데이터(스팸/피드백) 적재 |

## 🔄 주요 워크플로우

### 1. 스타일 학습 (최초 1회)
최근 발신 메일 200개를 분석하여 수신 유형과 작성 스타일을 시트의 `스타일_가이드` 탭에 기록합니다.

### 2. 메일 수집 및 필터링
* **검색 쿼리**: `2026-04-01` 이후 수신된 미읽음 메일 중 프로모션/소셜 카테고리는 제외합니다.
* **BCC 대응**: 수신인에 `alpha@kmong.com`이 없더라도, 본인 도메인(`kmong.com`)이 수신/참조인에 없는 경우 숨은 참조로 간주하여 처리 대상에 포함합니다.

### 3. 모델 라우팅 및 처리
* **Haiku (분류)**: 메일을 `SPAM`, `SOURCING`, `NORMAL`로 3단 분류합니다.
* **Spam**: 구글 시트 `스팸_보관함`에 기록 후 종료합니다.
* **Sourcing**: `sourcing` 라벨을 부착하고, 전문적인 확인 및 심층 질문이 포함된 초안을 작성합니다.
* **Normal**: 개인 스타일 가이드를 반영하여 정중한 비즈니스 답장 초안을 작성합니다.

### 4. 슬랙 보고 및 피드백
* 생성된 초안은 즉시 슬랙 DM으로 전송됩니다.
* 슬랙 메시지에는 Gmail로 즉시 이동할 수 있는 딥링크가 포함됩니다.
* 초안 데이터는 `프롬프트_피드백` 시트에 자동 적재되어 향후 프롬프트 개선에 활용됩니다.

## ⚠️ 주의사항
* **No Markdown**: AI가 작성한 티가 나지 않도록 초안에는 `###`, `**` 등 마크다운 형식을 일절 사용하지 않습니다.
* [cite_start]**보안**: API 키와 슬랙 Webhook URL은 반드시 `Script Properties`에 저장하여 관리합니다. [cite: 158]
* [cite_start]**실행 제한**: GAS 실행 시간 6분 제한을 고려하여 한 회차당 최대 10개의 메일만 처리합니다. [cite: 184]
